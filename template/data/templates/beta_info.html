What is beta:

This is a testing stage prior to pushing major changes to production. There may be new changes here on both the backend and frontend of the site - this is where they are all stress tested to ensure reliability and functionality.

Please note that the beta testing stage is volatile, and both safety and security of data cannot be guaranteed. You are signing up to the beta website at your own risk.

Besides, the only things we actually store are your Discord ID (public information), Minecraft username (public information) and whitelist state.

Additional Data Collection:

As this is a site under beta testing, we may conduct additional data collection at any time. We may collect this information to:
  • Gather user interest in specific features.
  • Diagnose and resolve errors in the website.
  • Contact a user if we require additional information about an error.
Please note that a developer will NOT ask for any "session tokens"/"session cookies". Anyone who requests such information should be reported directly to the real development team.

The following is a list of additional information that may be collected:
  • IP Address
  • User Agent of your Browser
  • Pages Accessed on this Site
  • Type of Request (GET, POST, HEAD etc.)
This data may be stored on disk while we work on the site. This will not be the case on the final website.
