ActionMC Skins Server

This website is equipped with a skin server which is desgined to bring skins and capes back to Minecraft 1.2.5. Unfortunately, you cannot use this skins server without performing some modifications first.

There are two ways to use our skin server:

Hosts File Modification (advised for older computers)

This method requires you to modify a system file. We are not liable for damages you may cause by following this guide.
  1. Find your hosts file.

    For Windows users, this is located at C:/Windows/System32/drivers/etc/hosts (you may have to run Notepad as Administrator to edit this file).
    For Linux and macOS users, this is located at /etc/hosts.

    If your OS is not listed, seek advice from the site administration or use Google.

  2. Add the following line to your hosts file:

    107.21.82.253 s3.amazonaws.com

    This will redirect all requests to s3.amazonaws.com to our skins server.
    MAKE SURE TO REMOVE THIS LINE AFTER PLAYING AS IF YOU DON'T REMOVE IT, SOME OTHER WEBSITES OR GAMES MAY BREAK AS THEY MAY ALSO RELY ON AMAZON WEB SERVICES.

ActionMC JAR Mod (recommended for newer computers)

This method relies on replacing files inside Minecraft 1.2.5's JAR file in order to work.

It is advised to use MultiMC to modify the JAR file as it makes management of modifications much easier.

MultiMC Installation

  1. Install Minecraft 1.2.5 under MultiMC.

    You must have Minecraft 1.2.5 installed in a profile in MultiMC.


  2. Click "Edit Instance" and go to the Version tab.

    The version tab allows you to modify the JAR file in many ways such as adding JAR mods like this one or adding Minecraft Forge.


  3. Click "Add to Minecraft.jar" and browse to the ZIP file you just downloaded from GitHub.

    Do not extract the JAR file as you cannot import a folder. It must be in a ZIP file.


  4. Select it and then press Launch.

    The game should now launch with the ActionMC JAR Mod in place. You can verify this by checking the bottom left of the title screen and verifying that "ActionMC JAR Mod" is displayed.
