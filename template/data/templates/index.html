What we are:

We are a small server for a game called Minecraft, which is a game "about placing blocks while running from skeletons". In our case, we run a Creative Mode server which turns this game into what could be classed as video game LEGOs.

The server runs on a Mac Mini with a Core 2 Duo. We run the server with Java 8 on Adelie Linux for maximum performance gain. The server is hosted by Sean of Action Retro on his own personal Mac Mini.
